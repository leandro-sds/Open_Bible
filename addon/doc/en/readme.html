Open Bible (NVDA Add-on)

This add-on lets you read the Bible in NVDA using multiple versions (JSON files), with navigation by book → chapter → verse, search and productivity tools.

Main shortcut

  • Ctrl+Alt+B – Open the main Open Bible window

Features

  • Manage and switch Bible versions (JSON files)
  • Quick navigation by book, chapter and verse
  • Text search
  • Favorites and notes
  • Copy/share verses via the Windows clipboard

Bible files folder

Place JSON files inside biblias/ (without accents), next to the add-on data folder. Example filenames:

  • 01_ACF.json
  • 02_NVI.json
